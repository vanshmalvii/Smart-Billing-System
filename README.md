# Smart-Billing-System
This is a Python-based Billing System that allows users to generate bills, manage items, and calculate totals with date, time, and unique bill numbers.

🚀 Features :-

- Add multiple items with price and quantity
- Automatic bill number generation
- Date and time tracking
- Total bill calculation
- User-friendly console interface
- Organized data using lists and functions

🛠️ Technologies Used :-

- Python (Core Concepts)
  - Variables
  - Lists
  - Functions
  - Loops
  - Conditional Statements

🎯 Purpose of the Project :-

This project is created for learning purposes to understand how real-world applications like billing systems work. It also helps in improving Python fundamentals.
